TROUBLESHOOTING:
===========================================================

Problem: Red banner not showing
Solution: 
  1. Ensure backend is running on :8000
  2. Hard refresh browser (Ctrl+Shift+R)
  3. Check that cities have current_aqi > 200
  4. Check browser console for errors (F12)

Problem: Email not being sent
Solution:
  1. Verify Gmail credentials in .env
  2. Ensure app password is correct (without spaces)
  3. Check backend console for SMTP errors
  4. Verify "Less secure app access" is enabled in Gmail

Problem: WebSocket connection fails
Solution:
  1. Check backend is running on :8000
  2. Verify WebSocket URL is correct
  3. Check firewall isn't blocking port 8000
  4. Look for errors in browser console (F12)

Problem: Database connection fails
Solution:
  1. Verify PostgreSQL is running
  2. Check DATABASE_URL in .env
  3. Ensure database credentials are correct
  4. Verify port 5432 is accessible

Problem: Frontend shows blank page
Solution:
  1. Check npm packages installed (npm install)
  2. Verify backend is running on :8000
  3. Check browser console for errors (F12)
  4. Try hard refresh (Ctrl+Shift+R)
