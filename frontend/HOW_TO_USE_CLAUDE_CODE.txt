===========================================================
HOW TO USE CLAUDE CODE FOR FRONTEND ENHANCEMENT
===========================================================

STEP 1: OPEN CLAUDE CODE
=========================

In VS Code:
1. Press Ctrl+` (backtick) to open terminal
2. Type: ollama launch claude
3. Wait for Claude Code to launch
4. You'll see the Claude Code interface

STEP 2: COPY THE PROMPT
=======================

Open this file:
d:\Projects\terrapulse-ai\frontend\CLAUDE_CODE_PROMPT.txt

Copy the ENTIRE content (Ctrl+A, Ctrl+C)

STEP 3: PASTE INTO CLAUDE CODE
===============================

In Claude Code:
1. Paste the prompt (Ctrl+V)
2. Press Enter to send

Claude Code will read the prompt and understand:
- What to enhance (frontend only)
- What NOT to touch (backend, api.ts)
- Specific improvements needed
- Which files to modify
- What results are expected

STEP 4: LET CLAUDE CODE WORK
=============================

Claude Code will:
1. Analyze the frontend code
2. Identify files to modify
3. Make UI/UX improvements
4. Apply animations and styling
5. Maintain all functionality
6. Keep API calls unchanged

This will take 5-15 minutes depending on complexity.

STEP 5: VERIFY THE CHANGES
===========================

After Claude Code finishes:

In PowerShell:
1. cd d:\Projects\terrapulse-ai\frontend
2. npm run dev

In Browser:
1. Go to http://localhost:3000
2. Verify improvements:
   - Dashboard looks better
   - Alert banner is eye-catching
   - Animations are smooth
   - Mobile view looks good
   - No errors in console (F12)

If backend is not running, start it:
   cd d:\Projects\terrapulse-ai\backend
   uvicorn app.main:app --reload

STEP 6: TROUBLESHOOT IF NEEDED
===============================

If something breaks:
1. Check browser console (F12) for errors
2. Check that backend is running on :8000
3. Check that frontend is running on :3000
4. Verify npm run dev completes without errors

If changes didn't apply:
1. Save files (Ctrl+S in VS Code)
2. Browser refresh (Ctrl+R or F5)
3. Hard refresh browser (Ctrl+Shift+R)

If animations are broken:
1. Check tailwind.config.js was updated
2. Check src/styles/global.css was updated
3. Restart npm run dev
